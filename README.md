## Project Overview
SENADA is a mobile application prototype designed to help users learn and preserve indigenous languages through interactive lessons, vocabulary practice, and engaging activities. The platform focuses on making indigenous language learning accessible, interactive, and culturally meaningful.

This prototype was designed using Figma to demonstrate the user interface and overall learning experience of the application.

## Features
- Language lessons
- Translation
- Save favourite words & phrases
- Interactive quizzes and mini-games
- Stories and cultural learning content
- Connect with others through community

## Subscription Plans

### Basic Plan (Free)
- Access to language lessons

- Explore stories (read only)
- Games and quizzes
- Text translation
- Limited favourites

### Premium Plan (RM 9.90 / month)
Everything in the Basic Plan, plus:
- Voice and camera translation
- Word of the Day
- Explore stories through podcasts
- Unlimited favourites
- Bonus game coins
- Extra daily challenge coins
- Premium badge
